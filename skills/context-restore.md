# 스킬: 컨텍스트 복원 (Context Restore)

> 새 대화를 시작할 때 이전 상태를 빠르게 되살리는 스킬.

---

## 언제 쓰는가

- 새로운 대화 세션을 시작할 때
- 오랜 공백 후 프로젝트를 재개할 때
- 다른 사람에게 현재 상황을 브리핑할 때

---

## 실행 단계

1. `CLAUDE.md` 읽기 → 활성 프로젝트 목록 파악
2. 관련 `projects/[이름]/README.md` 읽기 → Compiled Truth 확인
3. `weekly/` 최신 파일 읽기 → 이번 주 맥락 파악
4. `raw/inbox.md` 확인 → 미분류 항목 점검
5. 브리핑 요약 작성 후 "어디서부터 시작할까요?" 질문

---

## 브리핑 출력 형식

```
## 현재 상황 브리핑 — [날짜]

### 활성 프로젝트
- [프로젝트명]: [한 줄 상태] → 다음 액션: [내용]

### 이번 주 맥락
- [weekly 파일의 핵심 요약]

### 미분류 항목
- inbox에 [N]개 항목 있음 / 없음

---
어디서부터 시작할까요?
```

---

## 주의사항

- Compiled Truth만 읽고 Timeline은 필요 시에만 확인
- 브리핑은 간결하게 (3분 안에 파악 가능한 수준)
- 추측하지 말고 파일에 있는 내용만 전달

---

## 관련 프롬프트

```
csp-brain을 읽고 현재 상황을 브리핑해줘.
```
