## 스킬 파일 작성 규칙

- 파일명: `[스킬명].md` (영문 소문자, 하이픈 구분)
- 템플릿: `_templates/` 에는 별도 템플릿 없음 — 아래 형식 사용

```markdown
# 스킬: [스킬명]

## 언제 쓰는가
(이 스킬이 필요한 상황)

## 실행 단계
1. 
2. 
3. 

## 주의사항
- 

## 예시
(구체적인 예시)
```
