## 목적 & 배경

- **왜 이 프로젝트를 시작했는가**: 엔지니어링 조직의 기술 역량을 정량화하고, 시간에 따른 변화(Temporal)를 시각화하여 최적의 육성 경로를 제시하기 위해.
- **해결하려는 문제**: 
  - 과거의 역량이 현재 어떻게 전이되었는지 추적 불가
  - 단순 리스트업된 스킬의 유기적 관계(Graph) 파악 부족
- **성공 기준**: 100명 이상의 엔지니어링 역량 맵 구축 및 개인별 맞춤 성장 로드맵 생성
