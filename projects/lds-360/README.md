## 목적 & 배경

- **왜 이 프로젝트를 시작했는가**: 기존의 단순 점수 방식 리더십 진단은 실질적인 행동 변화를 이끌어내는 페를 소마(Narrative)가 부족함.
- **해결하려는 문제**: 
  - 정량 데이터의 평면성을 극복하고 다각도의 정성적 분석 제공
  - 리더 본인이 수긍할 수 있는 '현상학적' 코칭 리포트 자동 생성
- **성공 기준**: 리더십 진단 후 코칭 만족도 및 실행 계획 수립률 향상
