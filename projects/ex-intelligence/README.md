## 목적 & 배경

- **Data Sensing Architecture**: 
  - **L1 (Declarative)**: 설문 등을 통한 선언적 데이터
  - **L2 (Behavioral)**: 활동 로그 등을 통한 행동 데이터
  - **L3 (Contextual)**: 텍스트 및 상황 맥락 데이터
- **해결하려는 문제**: 
  - HR 인사이트가 후행 지표 중심이라 선제적 개입이 어려운 구조
  - 이론적 추상화보다는 현상학적(Phenomenological) 현장 언어 중심의 출력 지향
- **주요 산출물**: 
  - 12-Cluster EX Full Inventory
  - HTML 동적 리포트 시스템
  - 정형/비정형 데이터 통합 리포팅 로드맵
